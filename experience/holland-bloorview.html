Holland Bloorview | PROPEL Lab.

Software Engineer


  • Conducted research and data collection.

  • Designed an algorithm for a mobile app that uses two ankle sensors to monitor a patient's gait. It streams gyroscopic and accelerometer data, performs matrix transformations to adjust for irregular sensor placement, and extracts gait metrics like stance-time symmetry.

  • Enhanced the algorithm's speed and enabled gait event detection regardless of sensor orientation, significantly improving usability.

  • Improved user experience across the Swift-UI app through design improvements and collecting user surveys.

  • Multiple clinics now use the app, allowing patients to monitor and improve their gait from home.
